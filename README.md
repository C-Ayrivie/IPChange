# IPChange
``` texte
1)Preambule:

Le projet à etait crée pour que mes colaborateurs pour qu'ils puissent changer l'ip des ordinateur de programmation sans avoir à ouvrir les paramétres et de façon interactive pour ne pas se tromper.

2)Paramétres:

Dans ce programme j'ai intégrer un fichier de paramétres pour pouvoir modifier simplement le nom de l'interface, le masque et l'adresse ip un cas par defaut a etait integrer.

-Dans le fichier de config:.
/!\ Pour le nom de l'interface à la ligne #1 et #4 il faut changer le contenue des guillemets seulement.

#1.ethernet="Ethernet"        | Nom de l'interface pour la premiére interface.
#2.eadrfixe=192.168.1.222     | IP pour la premiére interface.
#3.emasque=255.255.255.0      | Masque pour la premiére interface.

#4.wifi="WI-FI"               |
#5.wadrfixe=192.168.1.220     | Paramétres pour la seconde interface.
#6.wmasque=255.255.255.0      |

-Les paramétres par defaut:

#1.ethernet="Ethernet"        |
#2.eadrfixe=192.168.1.220     | Paramétres pour la premiére interface.
#3.emasque=255.255.255.0      |

#4.wifi="WI-FI"               |
#5.wadrfixe=192.168.1.220     | Paramétres pour la seconde interface.
#6.wmasque=255.255.255.0      |
```
